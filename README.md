# Leitura de Dados com Pandas 📊

Este repositório contém exemplos práticos de como realizar a leitura de arquivos nos formatos mais comuns utilizando a biblioteca **Pandas** em Python.

## 📁 Formatos abordados:

- CSV (.csv)
- Excel (.xlsx)
- SQL (banco de dados)
- JSON (.json)
- HTML (.html)
- XML (.xml)
